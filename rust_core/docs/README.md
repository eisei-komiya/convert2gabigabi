# convert2gabigabi Rust Core

## 概要

`convert2gabigabi` は画像を意図的に「ガビガビ」にするライブラリ＆CLIツールです。  
JPEG/PNG画像のファイルサイズを大幅に削減し、レトロでピクセル感のある画像に変換できます。

## 特徴

- **5段階のガビガビレベル**: 軽微(Lv1)から破壊レベル(Lv5)まで選択可能
- **高い圧縮率**: 最大95%以上のファイルサイズ削減
- **クロスプラットフォーム**: Windows/Linux/macOS対応
- **フォーマット統一**: 出力は全てJPEGに統一（最大圧縮効率）
- **直感的CLI**: オプションの柔軟な指定が可能
- **出力パス指定**: カスタムファイル名・パスで保存可能

## インストール

### 1. Rustの環境が必要
```bash
# Rustup経由でRustをインストール
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### 2. CLIツールのビルド＆インストール
```bash
cd rust_core
cargo build --release --bin gabigabi
cargo install --path . --bin gabigabi
```

## 使い方

### 基本的な使い方
```bash
gabigabi <入力ファイル> [オプション]
```

### 例

#### 最もシンプルな使い方（全てデフォルト）
```bash
gabigabi input.jpg
# -> 100%サイズ、レベル2、自動ファイル名で出力
```

#### ガビガビレベルのみ指定
```bash
gabigabi input.png -g 5
# -> 100%サイズ、レベル5（破壊級）
```

#### リサイズ率とレベル指定
```bash
gabigabi photo.jpeg -r 50 -g 3
# -> 50%サイズ、レベル3（重め）
```

#### 出力ファイルパス指定
```bash
gabigabi input.jpg -g 4 -r 75 -o /path/to/output.jpg
gabigabi image.png -o my_gabigabi.jpg
```

#### オプション順序は自由
```bash
gabigabi -g 4 -r 40 -o result.jpg image.png
gabigabi -o custom.jpg -r 60 image.jpg -g 3
```

### ヘルプ表示
```bash
gabigabi --help
```

## オプション詳細

| オプション | 説明 | デフォルト値 | 例 |
|------------|------|-------------|-----|
| `-r, --resize` | リサイズ率（%） | 100.0 | `-r 50` |
| `-g, --gabi` | ガビガビレベル（1-5） | 2 | `-g 5` |
| `-o, --output` | 出力ファイルパス | 自動生成 | `-o result.jpg` |

## ガビガビレベル詳細

| レベル | 名称 | JPEG品質 | 色数 | 効果 | 用途 |
|--------|------|----------|------|------|------|
| **1** | 軽微 | 40 | 16段階/ch | 軽い圧縮 | SNS投稿用 |
| **2** | 普通 | 20 | 8段階/ch | 標準的な圧縮 | 一般的な用途 |
| **3** | 重め | 10 | 4段階/ch | かなり圧縮 | 容量節約重視 |
| **4** | 極重 | 5 | 3段階/ch | 極度の圧縮 | ローエンド端末用 |
| **5** | 破壊 | 1 | 2段階/ch | 破壊的圧縮 | ネタ・芸術用 |

## 出力ファイル名

### 自動生成の場合
入力ファイルが `photo.png` の場合：
- レベル2: `photo_gabigabi_lv2.jpg`
- レベル5: `photo_gabigabi_lv5.jpg`

### `-o` オプション指定の場合
指定されたパス・ファイル名で保存されます。

## サポートする入力フォーマット

- **JPEG** (.jpg, .jpeg)
- **PNG** (.png)

※ 出力は全てJPEGになります（最大圧縮効率のため）

## 圧縮率の例

41KB のサンプル画像での圧縮率：

| 条件 | ファイルサイズ | 圧縮率 |
|------|---------------|--------|
| 100%・Lv2 | 11.2 KB | 73.1% |
| 50%・Lv1 | 5.6 KB | 86.5% |
| 50%・Lv2 | 4.3 KB | 89.5% |
| 50%・Lv3 | 3.1 KB | 92.5% |
| 50%・Lv4 | 2.6 KB | 93.7% |
| 50%・Lv5 | 1.8 KB | 95.7% |

## ライブラリとして使用

Rustコードから直接利用する場合：

```rust
use rust_core::resize_gabigabi;

// 画像データを50%、レベル3でガビガビ化
let input_data = std::fs::read("input.jpg")?;
let gabigabi_data = resize_gabigabi(&input_data, 50.0, 3)?;
std::fs::write("output_gabigabi.jpg", gabigabi_data)?;
```

## トラブルシューティング

### エラー: "リサイズ率は 0-100 の範囲で指定してください"
- `-r` オプションは1-100の範囲で指定してください

### エラー: "ガビガビレベルは 1-5 の範囲で指定してください"
- `-g` オプションは1-5の範囲で指定してください

### エラー: "入力ファイルが読み込めません"
- ファイルパスが正しいか確認してください
- JPEG/PNG以外の形式はサポートしていません

### エラー: "出力ファイルの保存に失敗しました"
- 出力ディレクトリの書き込み権限を確認してください
- ディスク容量が十分か確認してください

### ヘルプが表示されない場合
```bash
gabigabi --help
```
